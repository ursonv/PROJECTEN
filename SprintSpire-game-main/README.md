# SprintSpire-game

References: 

- Fall Guys Character (https://sketchfab.com/search?q=fall+guys&type=models)
- ObstacleFlipper (https://sketchfab.com/search?q=fall+guys&type=models)


Hoe speel je de game ?:
1. Vul je naam in en druk op play.
2. De game start binnen de 3 seconden...
3. Loop met je character naar de finish je kan de: (zqsd) toetsen, de pijltjes toetsen of de muis klik gebruiken en je kan shift gebruiken om te sprinten...
4. Wanneer je bij de finish komt keer je terug naar het beginscherm en ziet u uw score in het leaderboard.

Start project: 

1. npm i
2. yarn dev
